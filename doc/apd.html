This chapter presents a chemical database manager for handling chemical databases with MySQL and Mychem.

D.1. Introduction

The Mychem software is useful when working with chemical databases. In order to facilitate the creation and the management of such databases, mychemdb-manager.py, a Python program, has been added to this project. This script is a command line interface that permits to create or update chemical databases. A graphical version of this program will be provided by the ChemiSQL project in a near future. The graphical interface will be implemented in Java and will support another chemical cartridge projects, like OrChem or pgchem::tigress.
