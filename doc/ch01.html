1.1. Mychem

Mychem is a chemoinformatics extension for MySQL. This extension provides a set of functions that handles chemical data within the MySQL database. Mychem is based on Open Babel (version >= 2.3.0). As Open Babel is a well-known software in chemoinformatics, Mychem proposes reliable and fast functions. These functions permit you to search, analyze and convert chemical data. More informations about Mychem are available on the Mychem website.
