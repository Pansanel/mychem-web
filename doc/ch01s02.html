1.2. MySQL

MySQL is one of the most popular Open Source SQL database servers. It is a very fast, multi-threaded multi-user and robust application. MySQL Server is intended for mission-critical, heavy-load production systems as well as for embedding into mass-deployed software. Further informations about MySQL can be founded on the MySQL website.
